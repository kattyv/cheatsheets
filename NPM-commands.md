# NPM Commands

1. Installing dependencies from **package.json**:

```
npm install
```

2. Common **shorthands**:

  ```
  #install
 npm i <package-name>
 
 #uninstall
 npm un <package-name>
 
 #update
 npm up <package-name>
  ```
  
  3. **Flags** for npm install:
  
  ```
  # save as dev dependency (Package will appear in devDependencies)
  --save-dev or -D 
  ```
